Siga o link a seguir como critério para ver as versões de cada software para instalação:
https://www.tensorflow.org/install/gpu (analise os "Software requirements")

OBS: o tensorflow recomenda a partir de uma determinada versao para cadas sofware,
é recomendado pegar sempre a menor versão para evitar incompatibilidade. Ex: >=7.6 use

 	->CUPTI (Não precisa baixar)

		.baixa e copia a pasta cuda para o ultimo caminho recomendado no "Windows setup"
			-SET PATH=C:\tools
		.Abra o "Edit the system environment variables" (vamos adicioar novas variaveis de endereço)
			-Edit the system environment variables->Advanced->Environment variables
		.Em "System variables" de Edit... no Path
		.Clica "New" e acidiona os caminhos do link nas variaveis
			-Adiciona os 4 path do link, o primeiro se você ja tiver nas variaveis não precisa
	
	->Tensor RT (seleicone a versao MINIMA recomendada que seja GA)
		.Joga no mesmo diretorio do CUDA
			-SET PATH=C:\tools
		.Copia o endereço do TensorRT e adiciona as variaveis do path tbm como no item anterior

=================================2-Associando a GPU com o tensowflow e keras================================
A PARTIR DE 7:55 
	->Download anaconda:"https://www.anaconda.com/products/individual"
		.Caso não deseje baixar o mini conda também é possivel
	->Abra o cmd
		.Digite:
			conda (para ver se foi devidamente instalado)
			conda create -n <nome_do_ambiente_virtual> python=<versao do python ex:3.7>	(cria um ambiente virtual para armazenar suas libs e distribuicoes, é possivel ter varias libs em versoes diversas se cada uma estiver em um ambiente virtual)
			activate <nome_do_ambiente_virtual>	(ativa o ambiente virtual para podemos navegar e utilizado
			pip install --ignore-installed --upgrade tensorflow-gpu		(instala o tensorflow gpu nesse ambiente virtual
			pip install keras	(intala keras)
			pip install pandas	(instala pandas

	->agora basta utilizar o tensorflow nesse ambiente virtual,temos duas maneiras de acessarmos esse ambiente virtual
		.Pelo cmd:
			activate <nome_do_ambiente_virtual>
		.Pelo anacondas
			no canto superior esquedo terá a opção:
				Applications on "base(root)"
				Clique na seta do "base(root)"				
				Troque para-><nome_do_ambiente_virtual>
